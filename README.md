# DevProdMed – Exercices Laravel

Projet réalisé dans le cadre du cours **DevProdMed**.

Ce repository regroupe l’ensemble des exercices développés tout au long du semestre, principalement avec le framework Laravel.

Ce cours s’inscrit dans le cadre du **Bachelor en Ingénierie des Médias** à la **HEIG-VD (Haute École d’Ingénierie et de Gestion du Canton de Vaud)**.
